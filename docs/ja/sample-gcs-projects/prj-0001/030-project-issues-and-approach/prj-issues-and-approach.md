# 課題と解決アプローチ

## 1. 主要課題

1. 人気商品の欠品が多い
2. 仕入れ判断が忙しさや体調に左右される
3. つけ記録が散在し、記録漏れが発生する
4. 店主以外が店番すると状況把握が難しい

## 2. 課題別アプローチ

### 2.1. 欠品の増加

- 原因:
  - 在庫数と売れ行き傾向を把握できていない
  - 発注タイミングが商品ごとにばらついている
- 解決アプローチ:
  - 商品ごとに発注点と定番商品フラグを設定
  - 在庫が発注点を下回ったら発注候補一覧を表示

### 2.2. つけ記録の漏れ

- 原因:
  - ノート運用で書き忘れ、読み間違いが起こる
  - 店番交代時に記録ルールが徹底されない
- 解決アプローチ:
  - 顧客ごとのつけ残高をシステム管理
  - 販売登録時に現金またはつけの選択を必須化
  - 顧客名、ニックネームでの検索をサポート

## 3. 全体方針

- 現場の使い方に寄せた画面と用語を優先する
- 複雑な分析より、毎日見る在庫と売上の可視化を優先する
- 将来拡張を見据えつつ、初期リリースはシンプルにする
