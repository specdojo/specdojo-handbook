# プロジェクト概要

## 1. 背景

地域の子どもたちに長年親しまれている駄菓子屋きぬやでは、販売記録や在庫管理をノートと記憶に頼って運用しています。

- 人気商品の欠品が増えてきた
- 仕入れタイミングと数量が勘頼りで在庫のムラが大きい
- 常連のつけ管理が曖昧で記録漏れが起きやすい

## 2. 目的

- 駄菓子の販売、在庫、顧客情報をシンプルに一元管理する
- 店主一人でも無理なく使える画面と操作フローを提供する
- 欠品と過剰在庫を減らし、店舗運営の継続性を高める

## 3. 期待効果

- 人気商品の欠品回数を月間 50% 削減
- 仕入れ金額のムダを年間 20% 削減
- つけ残高を可視化し、会計トラブルを防止
