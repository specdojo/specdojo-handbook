# 品質管理計画 作成指示

## 1. 目的と前提

- 目的: 品質目標・レビュー方針・品質基準を定義する。
- 参照ルール: `../rulebooks/pm-quality-management-plan-rulebook.md`
- 主な内容: 品質目標、レビュープロセス、メトリクス、検査基準、是正。

## 2. 入力情報

- 品質目標と達成期限
- レビュー計画（種別、頻度、実施者）
- 品質メトリクス（算出方法、閾値、報告先）
- 検査基準と是正プロセス

## 3. 出力フォーマット

- Frontmatter は `id` / `type` / `status` を必須とする。
- `id` は `pm-quality-management-plan`、`type` は `project` を使用する。
- 本文は次の見出し順で作成する。

1. 概要
2. 品質目標
3. レビュープロセス
4. 品質メトリクス
5. 検査基準と是正
6. 関連ドキュメント

## 4. 記述ルール

- 品質目標は定量指標または判定基準を伴って記述する。
- メトリクスは算出方法と閾値をセットで記述する。
- 不適合時の是正と再判定条件を記述する。

## 5. 禁止事項

- 定性的な目標のみを記載しない。
- 算出方法なしで指標だけを記載しない。
- 是正プロセスを未定義にしない。

## 6. 最終チェック

- 必須見出しが欠落していない。
- 品質目標、閾値、是正条件が記載されている。
- 関連ドキュメントが空欄でない（`（本書のみ）` 可）。
