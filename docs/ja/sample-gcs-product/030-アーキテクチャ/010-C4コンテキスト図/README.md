# C4コンテキスト図

システム全体の関係者・外部システム・対象システムの関係を、C4コンテキスト図として定義します。

## 記法ガイド

1. 図を含むドキュメント（Markdown）の作成・更新ルールは [cxd-rulebook.md](../../../specdojo/rulebooks/cxd-rulebook.md) を参照してください。
2. 生成AIを活用して Mermaid 図を含むドキュメント(Markdown)を作成する場合は [cxd-instruction.md](../../../specdojo/instructions/cxd-instruction.md) をプロンプトのテンプレートとして利用してください。
3. Mermaid 図の作図ルールは [cxd-mermaid-rulebook.md](../../../specdojo/rulebooks/cxd-mermaid-rulebook.md) を参照してください。
4. 生成AIを活用して Mermaid の図を作成する場合は [cxd-mermaid-instruction.md](../../../specdojo/instructions/cxd-mermaid-instruction.md) をプロンプトのテンプレートとして利用してください。
